One easy task that anyone can do is look at this page and let me know if any of these picks have since come true.

This page is built with TiddlyWiki, which is a nifty single page wiki tool with lots of options for taking little bits of information and composing them together in different ways. When I realized that, my mind jumped to the many, many little bits of information in The Rickies, and I just had to try it out.

As a single page wiki, the page URL doesn't change as items are opened. To link to specific items, use the menu in the top right of each item and select "permalink" to create a link to just that item, or "permaview" to create a link to all items currently open. There is also a "static link" which is a lightweight, JavaScript-free version of the item.
